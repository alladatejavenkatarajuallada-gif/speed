# Experiment 1: JavaScript Basics

This experiment demonstrates basic JavaScript operations including console logging, alert boxes, and array manipulations.

## How to Run

1. Open the `index.html` file in any modern web browser.
2. Check the browser console to see the "Hello, World!" message.
3. Click the "Calculate Sum" button to see an alert with the sum of two numbers.
4. The array operations will be displayed automatically on the page.

## Features

### Part A
- Console logging "Hello, World!"
- Calculating sum of two numbers and displaying in an alert box

### Part B
- Creating an array of 5 cities
- Displaying the total number of cities
- Adding a new city at the end
- Removing the first city
- Finding and logging the index of a specific city
